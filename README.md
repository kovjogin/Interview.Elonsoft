# elonsoft.chat

## Задание
Реализовать простой чат без бэкенда с использованием Vue.js. Функционал чата: пользователь вводит свой логин в форме и попадает в общий чат, где может читать сообщения всех пользователей и писать свои.

### Требования
- Сделать чат в стиле Material Design. В качестве примера можно взять макет: 
- Перед входом в чат пользователю предлагается ввести свое имя, которое в дальнейшем используется для идентификации его сообщений. Сохранять авторизацию в localStorage. При перезагрузке страницы авторизация не теряется.
- При входе в чат пользователь видит все ранее написанные сообщения.
- Добавить возможность выхода и входа под другим пользователем с новым именем. 
- Реализовать сохранение сообщений локально (localStorage). При перезагрузке страницы сообщения не теряются.
- Дополнительно: реализовать редактирование и удаление своих сообщений.
